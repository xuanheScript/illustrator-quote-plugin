## 🔄 发布流程

1. **准备阶段**
   ```bash
   # 安装依赖
   npm install zxp-sign-cmd --save-dev
   
   # 运行完整测试
   npm run lint
   npm run build:cep
   ```

2. **构建阶段**
   ```bash
   # 创建发布版本
   npm run release
   ```

3. **验证阶段**
   - 使用 Anastasiy's Extension Manager 测试安装
   - 在不同 AI 版本中验证功能
   - 确认导出的 CSV 格式正确

4. **发布阶段**
   - 上传 ZXP 文件到发布平台
   - 发布安装说明文档
   - 更新项目主页
